# Hackathon-Manuel-Healthcare
 Código para monitoramento de frequência cardíaca, queda e presença
 
Código responsável por adquirir os dados do sensor de frequência cardíaca SM-S4303R, sensor de vibração SW-18015 e sensor de presença DYP-ME003, realizar o tratamento destes dados e transmití-los via serial RS232.
O código roda encima de um Arduíno UNO e foi desenvolvido durante o Hackathon Manuel em 2019.
